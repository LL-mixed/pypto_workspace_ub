# Simulator Archive

This directory keeps non-active simulator components out of the main working paths.

Current archive sets:
- `qemu-device-legacy/`: legacy `linqu-ub` QEMU-device stack (`qemu` + `qemu-device`)
- `guest-probe-legacy/`: legacy bare-metal guest probe for `linqu-ub`

Active dual-node development/testing uses:
- `simulator/vendor/qemu_8.2.0_ub`
